# Contributing

Thanks for your interest in contributing! This project is maintained by the Center for Open Science (COS) and partners.

We welcome contributions via pull requests. Please follow standard GitHub practices: fork the repo, use a feature branch, and open a clear, respectful PR. By contributing, you agree to license your work under the Apache 2.0 License.

Questions or feedback? Open an issue or contact the maintainers directly.

We’re glad you’re here.
